## DR Utility Classes

A library with everyday use classes and methods:
* `Arrays`
  - `first`  
  - `firstOrNull`  
  - `last`  
  - `lastOrNull`  
  - `tryFind`  
  - `mapAssoc`  
  - `reindex`  
  - `remove`  
  - `search`  
  - `unique`  
  - `diff`  
* `Assert`
  - `notNull`
  - `isArray`
  - `isCallable`
  - `isInt`
  - `isString`
  - `notFalse`
  - `instanceOf`

## Installation
```shell
composer require digitalrevolution/utils
```
